# Crates

This directory is reserved for extracted Rust crates that will replace the current monolithic networking layout.

Planned crates:

- `bp-protocol`
- `bp-signaling`
- `bp-webrtc`
- `bp-yggstack`
- `bp-proxy`
- `bp-observability`

The migration has been documented first to keep the current project buildable while the next transport rewrite is prepared.

# Пошаговый план реализации: Авторизация Google OAuth в Minecraft P2P Connector

**Дата**: 2026-05-20  
**Автор**: Antigravity  
**Статус**: READY TO EXECUTE

Этот план описывает последовательные шаги по доведению авторизации Google OAuth до идеального автоматического состояния, используя прямой поток авторизации (через порт 14235).

---

## Шаг 1. Верификация готовности окружения (Supabase)
* [x] **ПодтвердитьRedirect URL**: Убедиться, что `http://localhost:14235/callback` успешно добавлен в список Redirect URLs в панели Supabase Dashboard (выполнено пользователем).

## Шаг 2. Разработка и адаптация фронтенда (`src/main.js`)
* [ ] **Проверить обработчик `#btn-google`**:
  * Убедиться, что при нажатии вызывается Tauri команда `start_oauth_server` на Rust-бэкенде.
  * Убедиться, что открывается точная прямая ссылка:
    `https://mjbqlrzcijxlontrbhak.supabase.co/auth/v1/authorize?provider=google&redirect_to=http://localhost:14235/callback`
  * Убедиться, что кнопка переходит в состояние загрузки ("Ожидание авторизации в браузере...").
* [ ] **Проверить слушатель событий `oauth-login`**:
  * Убедиться, что слушатель регистрируется при инициализации в функции `initAuth()`.
  * При получении события `oauth-login` извлекаются `access_token` и `refresh_token`, и вызывается метод `supabase.auth.setSession(...)`.
  * При успехе вызывается `populateProfilePanel()` и оверлей закрывается.
  * При ошибке выводится сообщение об ошибке в UI.

## Шаг 3. Локальная сборка и тестирование компиляции
* [ ] Запустить `cargo check` в директории `src-tauri` для валидации Rust-кода.
* [ ] Запустить `npm run tauri dev` для старта приложения в режиме разработки.

* [ ] Нажать кнопку "Войти через Google" в приложении.
* [ ] Проверить запуск внешнего браузера с правильной ссылкой.
* [ ] Пройти авторизацию в Google-аккаунте в браузере.
* [ ] Проверить перенаправление на `http://localhost:14235/callback`.
* [ ] Убедиться, что браузер отображает красивую OLED-страницу успешного входа.
* [ ] Убедиться, что Tauri-приложение мгновенно и автоматически авторизовалось, скрыло окно входа и отобразило профиль пользователя.

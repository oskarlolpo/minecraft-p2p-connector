# Итоговое резюме попытки релиза v0.3.24
Дата: 2026-04-12

Данный файл содержит описание функций, которые планировалось внедрить в версии v0.3.24. В связи с техническими сложностями и регрессиями было принято решение откатиться к v0.3.23.

## Запланированные улучшения:

1. Автоматизация проброса портов (UPnP)
   - Использование библиотеки `igd-next` для автоматического маппинга портов на роутерах.
   - Цель: избавить пользователей от ручной настройки маршрутизатора.

2. Мониторинг трафика в реальном времени
   - Расчет скорости передачи данных (BPS) для каждого пира.
   - Визуализация трафика через Canvas-графики в интерфейсе.

3. Системные уведомления Windows
   - Использование `tauri-plugin-notification` для оповещения о подключении и отключении игроков.
   - Цель: информировать хоста, даже если окно приложения свернуто.

4. Дизайнерское обновление (UI Overhaul)
   - Переход на Glassmorphism дизайн.
   - Новый макет Dashboard с отображением статистики.

## Причины отката:
- Ошибки компиляции Rust (ownership issues, type mismatches).
- Недоработанный интерфейс (UI feels unfinished).
- Регрессии в стабильности сетевого менеджера.

## Рекомендации для будущей реализации:
- Использовать клонирование AppHandle для асинхронных уведомлений.
- Проверять сигнатуры библиотек Rust (igd-next) перед использованием.
- Внедрять UI изменения итеративно, а не целиком за один релиз.
